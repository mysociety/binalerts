You can bulk-update bin collection data by uploading a file containing street names and days.
Currently, two types of import file are supported:

  • XML file generated by running http://www.barnet.gov.uk/garden-and-kitchen-waste-collection-streets.pdf through "pdftohtml -xml"
  • a comma-separated file in one of two formats. Note both must contain the titles line as shown:
    • "native" format: one row per collection:
      street,postcode,type,days
      Shetland Street,NW1,D,Tuesday
    • "day columns" format (used by Barnet): listing only street names, Mon-Fri:
      Monday,Tuesday,Wednesday,Thursday,Friday
      Horse Street,Dog Road,Cat Close,Donkey Drive,Llama Lane
      ,Panda Place,,,Cobra Crescent

Once you've uploaded the file(s), apply the Import data from file action to it to import the data.

The file is deleted once the import has been completed.
